## 🚀 Как запускать

Для сборки и запуска любой лабораторной работы выполните следующие команды:

1. Перейдите в папку с работой:
    ```bash
    cd lab_xx
    ```

2. Соберите и запустите через make:
    ```bash
    make run
    ```
